Οι οδηγίες αυτές αναφέρονται σε περιβάλλον linux και συγκεκριμμένα χρησιμοποιήθηκε το 
λειτουργικό σύστημα Ubuntu 20.04.
Αρχικά θα κάνουμε install κάποια εργαλεία απαραίτητα στην υλοποίησή μας.
Το πρώυτο installation θα είναι για το geth για το οποίο θα τρέξουμε στο terminal τις παρακάτω εντολές 
$sudo apt install software-properties-common
$sudo add-apt-repository -y ppa:ethereum/ethereum
$sudo apt-get update
$sudo apt-get install ethereum
Έπειτα θα χρειαστούμε το NodeJS και το NPM
Πρώτα θα ελέγξουμε αν τα έχουμε ήδη εγκατεστημένα στον υπολογιστή μας, με τις εντολές
$node -v  και 
$npm -v
Αν τα έχουμε εγκατεστημένα θα πρέπει να επιβεβαιώσουμε ότι η έκδοση του NodeJS είναι τουλάχιστον 8
και η έκδοση του NPM είναι τουλάχιστον 5.
Αν δεν είναι ήδη εγκατεστημμένα θα τα εγκαταστήσουμε τρέχοντας τις εντολές
$sudo apt install nodejs 
$sudo apt install npm
Τέλος θα εγκαταστήσουμε το Truffle. Για την εργασία χρησιμοποιήθηκε η τέταρτη έκδοση του Truffle
Τρέχουμε τις παρακάτω εντολές
$sudo npm install -g truffle@4.0.4
Αφού εγκαταστήσαμε τα απαραίτητα εργαλείαθα αρχίσουμε να ασχολούμαστε με το node που έχει υποβληθεί.
Στον φάκελο private θα βρούμε όλα όσα χρειαζόμαστε για το node. Το node που έχουμε ορίσει για το 
blockchain μας περιέχει τρία account τα hash των οποίων μπορούν να βρεθούν στον φάκελο keystore.
Το account 0, είναι ο miner. Για να ξεκινήσουμε τη διαδικασία θα κάνουμε πρώτα το αρχείο startnode.sh
εκτελέσιμο. Για να γίνει αυτό θα τρέξουμε την παρακάτω εντολή στον φάκελο private
$chmod +x startnode.sh 
και έπειτα θα τρέξουμε το αρχείο με την εντολή
$./startnode.sh
Την πρώτη φορά που θα το τρέξουμε ίσως χρειαστεί αρκετή ώρα. Έπειτα θα ξεκινάει κατευθείαν. 
Ενώ το πρόγραμμα τρέχει ανοίγουμε ένα δεύτερο terminal και τρέχουμε την παρακάτω εντολή.
$geth attach
Από εκεί μπορούμε να κάνουμε συναλλαγές με τα accounts που έχουμε φτιάξει. Λεπτομερής οδηγός με 
τις διαθέσιμες εντολές μπορεί να βρεθεί στο παρακάτω link
https://geth.ethereum.org/docs/interface/javascript-console
Για τις εντολές που απαιτούν κωδικό, ο κωδικός μπορεί να βρεθεί στο αρχείο password.sec και
είναι pass 1234.
Μπορούμε να βγούμε από το geth console πλκτρολογώντας exit και να σταματήσουμε το πρόγραμμα με ctrl+c.

Τώρα θα ασχοληθούμε με την εφαρμογή smart contracts στο node που φτιάξαμε 
Αρχικά στην υλοποίησ μας έχουμε δύο smart contracts. Το ένα λέγεται migration και είναι απαραίτητο και 
μας επιτρέπει να την εφαρμόζουμε σε διάφορα nodes και το άλλο λέγεται Greetings και είναι ένα 
απλό παράδειγμα που μας επιτρέπει να θέτουμε και να τυπώνουμε ένα μήνυμα.
Για να εφαρμόσουμε τα smart contracts στο node που φτιάξαμε στο προηγούμενο βήμα, ανοίγουμε το 
terminal στον φάκελο "Smart Contracts" και πληκτρολογούμε την παρακάτω εντολή
$truffle migrate --compile-all --reset --network mynet
Η εντολή αυτή θα μας κάνει compile. Έπειτα τρέχουμε την εντολή
$truffle console --network mynet
για να μπούμε στο truffle console όπου μπορούμε να γράφουμε εντολές σχετικά το node και τα 
smart contracts. Επιπλέον για να τρέξουμε τις συναρτήσεις που αποτελούν το smart contract Greetings 
τρέχουμε στο console την εντολή
$Greetings.deployed().then(function(instance){app = instance;})  
Αφού την τρέξουμε μπορούμε για παράδειγμα να τρέξουμετην εντολή
$app.getGreetings()
που θα μας εμφανίσει το μήνυμα του contract ή να τρέξουμε την εντολή
$app.setGreetings("Hello Friends!", {from: web3.eth.accounts[0]})
που θα αλλάξει το μήνυμα σε "Hello Friends!"
Μπορούμε πάντα να βγούμε από το console με την εντολή 
$.exit
